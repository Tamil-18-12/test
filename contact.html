About the App

This is a comprehensive booking management application built entirely with client-side technologies (HTML, CSS, and JavaScript). It features separate admin panels for creating and managing events—such as catering, travel, or photography packages—and beautiful, client-facing pages for users to view and book these services. All data, from event details to customer bookings and reviews, is persistently stored in the browser's local storage, creating a seamless, database-like experience without requiring a backend server.
